What I learned: Aug. 16 - Aug. 22, 2022

This week I've been finishing my preparations with my equipment to start with my new client. I’ve learned more about the internal procedures the company takes when recruiting a developer. Aside from that, this week my learning has been focused on understanding React. I've been following a course where I’ve learned more about the react components, understanding how JSX works and why some approaches are more helpful depending on what you want to build.

Following along with the course, I’ve also been able to see how important and useful the reusability of code can be. And with the practice I’ve been getting from the course, I could also see how leaving parameters instead of hard coding some components allows for better reusability. I’ve completed some projects in the course that left me with a better understanding of this reusability, like the expenses trackers, and the user list page, you can find my advance of the project here

Right now, I can understand the useState and useEffect react hooks, I think that there’s still more practice for me to be done to fully grasp them. Also, with the useReduce hook, that I learned about today, I see how important it is and how useful it could be but I’m still struggling to fully understand it. Finally, I think this course is helping me feel more confident in navigating among different files and seeing the flow in which components’ information is being passed around.
