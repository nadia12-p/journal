JavaScript & React

This week I got my job offer with Encora and I will be starting with my client on the 22nd., I’ve been sorting out my required documents for the position and also been learning more about JavaScript and React for the job. I've been clearing up some topics of JavaScript that came up in my interview that I wasn’t too sure about like arrow functions, but also been studying more things like the fetch API, async/await, and been getting used to the syntax. I’ve been focusing mostly on JavaScript because I was confused when I first started a react course without better understanding of how JavaScript works and was confused by how the framework works but also because of the base language it has. This following week, I’m sure I won’t be as confused as I was months before with learning React thanks to understanding JavaScript better.
